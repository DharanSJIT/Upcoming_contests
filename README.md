# Upcoming Contests - CodePrepPro

A professional contest carousel platform designed to connect talented students with top-tier companies through competitive programming excellence.

## Features

- **Auto-rotating Contest Carousel**: Displays upcoming contests from major platforms
- **Platform Filtering**: Filter contests by LeetCode, Codeforces, CodeChef, AtCoder, HackerRank
- **Professional Design**: Built for sponsors and industry partnerships
- **Responsive UI**: Works seamlessly on desktop and mobile devices
- **Real-time Updates**: Live contest information with registration links

## Tech Stack

- React 18
- Framer Motion (animations)
- Tailwind CSS (styling)
- JavaScript ES6+

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/DharanSJIT/Upcoming_contests.git
cd Upcoming_contests
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Project Structure

```
src/
├── App.js              # Main app component
├── App.css             # Tailwind CSS imports
├── ContestCarousel.jsx # Main contest carousel component
└── index.js            # React entry point
```
